# Genesis

A random tile map generator for use in HTML games.

## Quickstart

```
$ npm install
$ npm install -g cake
$ cake serve
```

View demo site at http://localhost:8000

## Examples

Parameters like size and distribution of land are configurable, so you can have one large land-mass or a lot
of small islands.
